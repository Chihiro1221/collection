# 收集箱

打算做一些ts的小玩意然后收集成一个网站，等之后多起来的时候在上到线上。

## 技术选型
### 开发语言
- JavaScript
- HTML5
- CSS3
### 开发框架
- Vue3
- Vite3
- TailwindCSS
- Ant Design
### 语法支持
- TypeScript
- SCSS
### 代码检测
- ESLint
- Prettier
- StyleLint

## 开始使用
1. 克隆项目
```bash
git clone https://github.com/Chihiro1221/collection.git
```
2. 安装依赖
```bash
pnpm install
```
3. 启动服务
```bash
# 端口开在4399
pnpm run dev 
```
4. 编译项目
```bash
pnpm run build
```
